Flask的request

Written by Song with ♥ on June 22, 2017 in Python

flask的request模块的属性

1、method：请求的方法（get或者post）

2、form：返回form的内容

3、args和values：args返回请求中的参数，values返回请求中的参数和form

4、cookies：cookies信息

5、headers：请求headers信息，返回的结果是个list

6、date、files：date是请求的数据，files随请求上传的文件
